Next Games Ease Index

Rules

The Next x Games Ease Index is equal to sum of the ease index values for each of X games. Single game index value is calculated as a sum of following two factors:

  • opponent table rank
  • game venue

Both factors values are determine as follows:

Opponent Rank {{ item.range }} Points {{ item.value }}
Venue {{ item.venue }} Value {{ item.points }}

Example

Let's imagine a team plays next 3 games:

  1. home vs 2nd
  2. away vs 8th
  3. home vs 17th

For the {{ 1 | numerals }} game the team gets 0pts for the opponent rank factor + 1pts because of playing home. So the Ease Game Index equals 1pts.
For the {{ 2 | numerals }} game the team gets 4pts (opponent rank) - 1pts (away game) = 3pts.
For the {{ 3 | numerals }} game the team gets 10pts (opponent rank) + 1pts (home game) = 11pts.
Summing up: 1 + 3 + 11 = 14. The Last 3 Games Ease Index equals 14;
